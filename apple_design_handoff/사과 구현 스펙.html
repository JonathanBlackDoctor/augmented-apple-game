캔디 글로스 — 사과 구현 스펙

확정된 캔디 글로스 사과를 완전히 동일하게 재현하기 위한 정밀 스펙입니다. 모든 좌표는 사과 바운딩박스를 0–100 으로 정규화했고, 그라데이션은 아래→위 페인트 순서로 기재했습니다. 광택(스페큘러)은 0 으로 확정, 밤 외곽선은 미사용입니다. 페이지의 모든 사과는 이 스펙으로 실시간 렌더된 실물입니다.

03 보디 그라데이션 레이어

아래에서 위로 3개 레이어를 합성합니다. hl(하늘 반사)과 amb(하단 음영)은 시간대에 따라 계산됩니다(05 참고).

/* 아래 → 위 (CSS는 첫 줄이 맨 위) */
3 ambient radial 54% 36% @50% 100%
        rgba(70,16,8,amb) → transparent 68%
2 sheen   radial 86% 78% @50% 2%
        hl → transparent 60%
1 base    linear 176deg
        top 4% · mid 54% · bot 100%

스페큘러(흰 점) = 0(미사용). 림 라이트는 04, 광원 계산은 05.

04 환경 림 라이트 (screen)

해·달 빛이 가장자리에 닿는 표현. screen 블렌드, 색은 해/달 디스크 색(rim), 불투명도는 rim_a(05). 어두울수록 강해집니다.

opacity: rim_a;  mix-blend-mode: screen;
radial 120% 95% @82% 80%  rim → transparent 40%
radial  90% 30% @50%  2%  rim → transparent 60%

05 낮밤 광원 모델

시간 t∈[0,1] 를 5 키프레임(R1=0 … R5=1) 사이로 선형 보간한 프레임 f 와 잠금 계수로 사과 변수를 계산합니다.

sky=0.6  rim=0.55  depth=0.35  gloss=0  nightOutline=false

hl       = mix(#ffffff, f.sky1, sky * 0.72)
amb      = 0.22 + depth*0.5 + f.star*0.16
rim      = f.disc            // 해/달 색
rim_a    = min(rim * (0.34 + 0.66*max(f.star,0.22)), 1)
shadow_a = 0.6 + depth*0.6
