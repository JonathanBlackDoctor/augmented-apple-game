# 배경 음악 (BGM) 파일

게임은 낮/밤 두 개의 루프를 동시에 재생하고, 하늘의 낮→밤 위상(`skyClock`의 `p`)에
맞춰 둘 사이를 **이퀄 파워 크로스페이드**합니다. (구현: `src/app/bgm.ts`)

여기에 아래 두 파일을 넣으면 코드 수정 없이 바로 재생됩니다.

| 파일명 | 역할 | 재생 구간 |
| --- | --- | --- |
| `bgm-day.mp3` | 낮 테마 (차분·따뜻) | 아침/낮·오후 (`p ≈ 0 ~ 0.42`, 그리고 동틀녘 `p ≈ 0.93 ~ 1`) |
| `bgm-night.mp3` | 밤 테마 (잔잔·포근) | 밤 (`p ≈ 0.62 ~ 0.88`) |

## 권장 사양

- **포맷**: MP3 (또는 `bgm.ts`의 경로를 `.ogg`/`.m4a`로 바꿔도 됨)
- **길이**: 30~90초. 제미나이는 보통 ~30초인데 그대로 써도 됩니다(아래 루프 설명 참고)
- **스타일**: 취향대로 — **차분·힐링** 또는 **밝고 경쾌** 모두 OK. 단 박자 충돌을
  피하려면 드럼은 빼거나(앰비언트) 아주 가벼운 퍼커션만 쓰고 낮/밤 곡 **템포를 동일하게**
  맞추세요
- **루프**: 클립의 **시작과 끝이 음악적으로 이어지게**(no hard ending / 잔향 꼬리). 재생은
  WebAudio `AudioBufferSourceNode(loop=true)`로 **샘플 단위 gapless** 처리되어 30초마다
  무음 끊김은 없습니다
- **음량**: 노멀라이즈된 상태로. 코드에서 마스터 볼륨 `0.5`가 곱해지므로 클리핑 없이 적당히
- **분위기**: 두 곡이 같은 키·악기 계열이어야 크로스페이드가 자연스럽습니다

### 30초가 짧게 느껴지면
같은 프롬프트로 2~3개 생성해 **이어 붙여** 60~90초 한 파일로 만들면 반복 주기가 길어져
덜 지루합니다(끝/시작 음만 맞추면 됨).

## 제미나이로 만들 때

`prompts/gemini-bgm.md`에 두 트랙용 음악 생성 프롬프트가 정리되어 있습니다.
생성한 파일을 위 표의 이름으로 이 폴더에 저장하세요.

> 파일이 없으면 게임은 그냥 무음으로 동작합니다(오류 없음).
