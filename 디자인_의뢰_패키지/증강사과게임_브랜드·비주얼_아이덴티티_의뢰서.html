바쁘면 이 절만.

한 줄 브랜드30초 안에 합 10 사과를 톡톡 터뜨리고, 증강으로 빌드를 키워 1:1로 겨루는 — 따뜻하지만 짜릿한, 손맛 좋은 경쟁 퍼즐.
가장 중요한 감각"손맛" — 드래그해서 사과가 펑 터지는 0.1초가 즉각적이고 만족스러워 보여야 함(브랜드의 1순위 정서).
두 번째 감각"성장의 화려함" — 증강이 쌓일수록 점수가 폭발하고 화면이 더 빛나는 위계(실버→골드→프리즘).
꼭 피할 것제네릭 AI 룩 — 의미 없는 보라/청록 그라데이션, 글래스모피즘 떡칠, 어디서나 본 둥근 카드. 이 게임만의 정체성을 원함.
꼭 지킬 것숫자 가독성(셀 ~20px에서도 1~9 즉독) · 사과게임 DNA(숫자 격자+드래그 박스) · 살아있는 하늘과의 공존.
자유컨셉·무드·컬러·사과 스타일·타이포·아이콘 양식·로고 — 전부 디자이너 제안. 현재 톤에 맞출 필요 0.

전권 위임의 의미

§03의 "절대 제약" 몇 가지만 지키면 비주얼 컨셉은 전적으로 디자이너의 제안을 따릅니다. 아래의 무드 영역(§04)·현재 토큰(§16)은 출발점/참고일 뿐, 정답이 아닙니다. 마음껏 갈아엎어 주세요. 역제안 환영.

01제품의 본질 & 각 표면이 짊어질 정서

브랜드는 "예쁨"이 아니라 정서의 운반입니다. 게임은 4개 기둥으로 서 있고, 표면마다 실어야 할 감정이 다릅니다.

#기둥이 정서를 가장 많이 지는 표면
1즉각적 손맛게임 보드·사과·선택 박스·제거 버스트·점수 팝업 — "톡 터지는" 즉각 만족.
2빌드의 쾌감증강 카드·티어 위계·보유 빌드·정산의 점수 폭발 — 성장·화려함.
3공정한 긴장대결 HUD·라운드 정산·MMR/티어·랭크 엠블럼 — 신뢰·승부의 무게.
4바이럴 성장결과/공유 카드·OG·데일리 챌린지·앱 아이콘 — "자랑하고 싶은" 매력.

비주얼·컬러·디테일의 무게를 이 순서대로. 1순위(손맛)가 흔들리면 나머지가 다 무너집니다.

꼭: 직접 손맛 체험

npm install && npm run dev → localhost:5173. 드래그→제거→콤보→증강을 직접 만져 30초의 리듬을 느끼셔야, "이 브랜드가 어떤 손맛을 그려야 하는지"가 잡힙니다.

03절대 제약 & 자유 영역

아래 "지켜야 할 것"만 빼면 전부 자유입니다.

지켜야 할 것 HARD

  • 사과게임 DNA: 숫자가 적힌 오브젝트 격자 + 드래그 박스 선택. 과일이 꼭 "사과"일 필요는 없으나, "숫자 오브젝트를 박스로 묶어 합을 만든다"는 구조는 유지.
  • 숫자 가독성 최우선: 모바일에서 셀 한 변이 ~20px까지 작아짐. 그 크기에서도 1~9가 즉시 읽혀야 하고, 빽빽한 170칸이 시각적으로 피로하지 않아야 함.
  • 반응형: 320px(모바일 세로) → 데스크톱. 보드는 항상 화면에 꽉 맞고, 터치·마우스 드래그 모두 자연스러워야 함.
  • 성능: 드래그 미리보기·파티클 60fps. 모바일 GPU를 의식 — 과한 blur/filter/대형 그림자 남발 금지.
  • 살아있는 하늘과 공존: 보드는 투명 캔버스로 낮→밤 하늘 위에 얹힙니다(§08). 하늘이 시간에 따라 변해도 사과·숫자·UI 가독성이 유지돼야 함(밤에도 또렷).
  • 두 "티어" 체계 구분: 증강 티어(실버·골드·프리즘)와 랭크 티어(아이언~마스터)는 다른 체계. 시각적으로 혼동 금지.
  • 한글 + 숫자 동시 가독.

완전 자유 YOUR CALL

전체 컬러/무드(다크·라이트·뭐든) 배경/하늘 컨셉·일러스트 사과/오브젝트 아트 스타일 타이포 선택 UI 컴포넌트 룩 증강·티어 아이콘 양식 이펙트·모션 언어 로고/워드마크 사운드 방향 제안(선택)

단 하나의 안티-패턴

특징 없는 "AI가 만든 것 같은" 룩 — 의미 없는 보라/청록 그라데이션, 글래스모피즘 떡칠, 어디서나 본 둥근 카드의 나열. 이 게임만의, 기억에 남는, 소유 가능한(ownable) 정체성을 원합니다.

04무드 방향 (출발점 영역 · 택1 아님)

방향을 고르라는 게 아니라, 반응을 끌어내기 위한 네 개의 출발 영역입니다. 취하거나·섞거나·거부하거나·완전히 새로 제안하세요. 각 팔레트는 예시입니다.

A. 햇살 과수원 (현재 임시 방향)

느낌 따뜻한 크림 종이 + 윤기나는 과수원 레드 + 꿀빛 골드. 낮밤 하늘과 자연스러움.
적합 손맛·친근. 리스크 안전하지만 평범해질 수 있음 — 더 밀어붙일 여지.

B. 한지 · 민화

느낌 한지 질감 + 민화 사과 + 먹선·단청색. 한국적이고 소유 가능.
적합 독창성·기억성. 리스크 작은 셀에서 질감·먹선의 가독성 관리 필요.

C. 네온 아케이드

느낌 다크 + 네온, 점수·콤보가 빛으로 터짐. 경쟁/랭크와 궁합.
적합 짜릿함·대결. 리스크 제네릭 게이밍 룩·다크 일변도 경계, 하늘 컨셉 재해석 필요.

D. 종이공예 (페이퍼크래프트)

느낌 레이어드 컷페이퍼 + 부드러운 그림자, 촉각적이고 다정. 제거=종이 조각 흩날림.
적합 손맛·캐주얼. 리스크 그림자 남발 시 성능 — 평면 그림자로 양식화.

※ 위 4안은 예시 좌표일 뿐입니다. 다섯 번째(예: 보석 세공/프리미엄 다크)를 제안하셔도 좋습니다.

유효(합10) 신호 데모

현재는 초록 채움+글로우 링으로 표현합니다. 37 ← 합 10 달성 순간. 이 "단 하나의 긍정 신호"를 새 시스템에서 더 또렷하고 황홀하게. 진행 중(합≠10)은 절제된 중립색.

다크/라이트는 한쪽만 해도, 둘 다 제안해도 됩니다(디자이너 판단). 단 토큰 구조로 — 색 하나가 DOM(CSS 변수)·Pixi 보드 양쪽에 파생되도록.

06타이포그래피 — 숫자가 주인공

이 게임에서 가장 많이 보이는 글자는 사과 위의 숫자 1~9입니다. 타이포는 곧 손맛의 일부.

숫자(히어로)170칸 × 매 라운드. 셀의 ~0.47–0.48 크기. 20–50px 전 구간에서 즉독. tabular(고정폭)·강한 웨이트. 점수/타이머에도 동일 숫자 정체성.
한글 본문현재 Pretendard. 버튼·설명·카피. 한글 전 음절 가독.
디스플레이현재 Newsreader/Gowun Batang(세리프) — 결과 큰 점수·제목 등. 브랜드 표정. 재선택 자유.
웹폰트한글 서브셋·로딩 전략(스플래시 연계). 라이선스 확인.

지금 짚을 불일치 하나

사과 핸드오프 스펙은 숫자를 Quicksand(확정, 폴백 Pretendard)로 정의했지만, 실제 보드 코드(theme.ts)는 Pretendard로 렌더 중입니다. 숫자 서체는 브랜드의 핵심 글리프이니 이번에 하나로 확정해 주세요(가독성 테스트 포함).

07사과 아트 디렉션 (최우선 브랜드 표면)

플레이 시간의 90%를 보는 오브젝트. PixiJS 캔버스에 그려지므로 벡터 스펙 또는 스프라이트로 납품(개발이 이식). 스타일은 완전히 재정의 가능합니다.

황금 보석 만능

▲ 현재 실루엣(viewBox 0–100)과 변형 색. "캔디 글로스" 방향이 별도 핸드오프(apple_design_handoff/)로 정리돼 있으나, 스타일 재정의는 환영합니다.

현재 스펙 요약 자산

실루엣 path 확정 · 소프트 코럴 톤(top #e8705a/mid #d4513a/bot #9c3a2a, linear 176°) · 광택(흰 점) 0 · 밤 외곽선 0 · 하늘 반사 하이라이트 + 하단 음영 + 림라이트(해/달 색)로 낮밤에 반응 · 줄기·잎.

변형 5종

normal 1~9 · golden(점수2배) · gem(+15) · bomb(+10) · wild(만능). 한눈에 구분되되 숫자 가독성·격자 통일감 유지. 빈 칸(제거됨)·보드 패널 처리도 포함.

요청

① 20–50px 양극단 가독성 ② "톡 터지는" 손맛이 정지 이미지에서도 느껴지는 형태/광택 언어 ③ 낮·밤 모두 또렷(하늘 반응 모델 유지 또는 재설계) ④ 변형 식별성↔통일감 ⑤ 제거 파티클의 모양·색이 사과 아트와 한 가족. 납품: 벡터 드로잉 스펙(색·치수·비율·광원식) 또는 스프라이트(SVG + @1/2/3x).

08살아있는 하늘 & 세계관 (시그니처 자산)

보드 뒤에는 게임 시계에 연동되는 이음매 없는 하루가 흐릅니다. 이건 이 게임의 가장 독창적인 브랜드 레버입니다 — 살리거나 재해석하되, 그냥 버리지 마세요.

아침 → 낮 → 해질녘 → 밤 → 황혼 → 다시 아침 (루프)
R1 아침/낮
R2 오후

작동위상 p∈[0,1] 루프(p=1≡p=0). 매치가 진행되며 라운드마다 시간대를 쓸고, 매치는 동틀녘에 끝나 깔끔히 아침으로 되돌아옴. 홈에선 60초 자동 순환.
시계 연동위상이 라운드 시계에서 파생 → 시간 증강(감속/정지)을 공짜로 상속, 증강 선택·일시정지 중엔 하늘도 멈춤. 브랜드가 게임 메카닉과 한 몸.
구성 요소해/달(반대편 호를 그림)·구름·과수원 지평선·먼/가까운 잔디·나무. 키프레임 7단계 보간.
제약순수 장식(보드/UI 뒤, aria-hidden). 어느 시간대에서도 사과·숫자·UI 가독성 사수. 모바일 성능.
